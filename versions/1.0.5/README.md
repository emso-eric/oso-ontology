## Overview

Version 1.0.5 further improves FAIR metadata,
provenance descriptions,
ontology engineering metadata,
semantic interoperability,
machine-readable accessibility,
and compatibility with FAIR assessment frameworks such as O’FAIRe.

---

## Main improvements in version 1.0.5

- Improved FAIR metadata compatibility
- Enhanced provenance and versioning metadata
- Improved ontology engineering methodology descriptions
- Extended DCAT and VoID metadata
- Improved machine-readable distributions
- Enhanced multilingual metadata
- Improved semantic interoperability
- Public SPARQL endpoint integration
- LOV indexing support
- Improved persistent identifiers and ontology references
